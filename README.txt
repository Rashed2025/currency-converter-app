Currency Converter App - A simple currency converter built with Streamlit and ExchangeRate-API. 
Convert between multiple currencies and view real-time exchange rates.

Setup:

Clone the repo and install dependencies.
Get an API key from ( ExchangeRate-API ) by signing up, verifying your email, and copying your API key.
Replace "YOUR_API_KEY" in the script with your actual key.
Run the app using streamlit run your_script.py.
Features: Convert currencies, fetch live exchange rates, and enjoy a clean interface.
