Currency Converter App

This is a simple currency converter application built using Streamlit and the ExchangeRate-API. The application allows users to convert between different currencies and view exchange rates against the US Dollar.

Features

Convert amounts between different currencies.

Displays the latest exchange rates.

User-friendly interface with a clean design.

Installation

Clone the repository and navigate to the project directory:

git clone <your-repo-url>
cd currency-converter-app

Install the required dependencies:

pip install -r requirements.txt

Obtain an API Key from ExchangeRate-API:

Visit https://www.exchangerate-api.com.

Sign up for a free account.

Verify your email address.

After verification, you will receive your API key.

Add your API key to the script:

Open the Python file.

Locate the line: API_KEY = "YOUR_API_KEY".

Replace YOUR_API_KEY with the API key you received.

Usage

To run the application, use the following command:

streamlit run your_script_name.py
